les ID

Ce paragraphe est identifié "identité_unique". Assez logiquement, une identité ne peut être utilisée que pour un élément html unique du document. Ceci permettra aussi de l'utiliser pour du JavaScript. L'élément "identité_unique" utilise la règle CSS appliquée à cet élément. C'est ce qui permet cette mise en forme
