l'héritage en CSS

Exemple de cascade CSS

Sous-titre 1

L'héritage nous permet de déterminer un style global sur les éléments parents, puis d'affiner les styles de certains des éléments enfants selon les besoins. Une telle approche rend l'écriture CSS très efficace.

Sous-titre 2

Le sous-titre de ce paragraphe est restylé en ligne. color: green; s'applique maintenant malgré l'héritage des règles aplliquées à body.
Cependant, les règles fusionnent c'est pour cette raison que text-align: center; et font-family: Helvetica; s'appliquent. Seule la nouvelle règle déterminée dans le ligne de h2 remplace la règle du parent.

Evidemment, l'héritage descend des parents aux enfants et ne remontent pas des enfants vers les parents !
